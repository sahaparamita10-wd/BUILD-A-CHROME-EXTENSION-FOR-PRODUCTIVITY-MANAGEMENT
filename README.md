*DESCRIPTION*: I create this project in Visual Studio Code. The Productivity Tracker Chrome Extension is a smart browser-based application designed to help users manage their time effectively while browsing the internet. In today’s digital world, people spend a large portion of their time online, often getting distracted by social media, entertainment platforms, and non-productive websites. This project aims to solve that problem by tracking browsing activity, analyzing productivity patterns, blocking distracting websites, and generating daily productivity reports. The extension is developed using JavaScript technologies such as Node.js and Express.js, making it efficient, scalable, and easy to maintain. The extension works directly inside the Chrome browser and continuously monitors the websites visited by the user. It records the amount of time spent on each website and categorizes websites into productive and non-productive groups. For example, educational websites, coding platforms, and work-related applications may be considered productive, while excessive usage of social media or video streaming websites may be marked as distracting. This data is then processed and stored using a backend server built with Node.js and Express.js. Node.js plays an important role in the project because it allows asynchronous handling of user data and browser events. Since browser activity generates continuous data, Node.js efficiently processes requests without slowing down the application. Express.js is used to create REST APIs that manage user information, browsing history, blocked websites, and productivity reports. The backend communicates with the Chrome extension to send and retrieve data in real time. One of the most important features of the project is website time tracking. The extension detects the active browser tab and calculates how much time the user spends on a specific website. This helps users understand where their time is being used. Many people are unaware of how much time they waste online every day. By providing detailed statistics, the extension encourages users to develop better digital habits and improve concentration. Another key feature is website blocking. Users can create a custom list of distracting websites that they want to avoid during study or work hours. Once a blocked website is opened, the extension prevents access and displays a warning message. This feature is especially useful for students preparing for exams, employees working remotely, and freelancers who need to stay focused. Some advanced implementations may also include a timer-based blocking system where websites are blocked only during specific hours of the day. The extension also generates daily productivity reports. These reports summarize browsing behavior, including total productive time, non-productive time, most visited websites, and productivity percentages. Reports can also motivate users by showing progress over time. The project has several real-world applications. In the education sector, students can use the extension to maintain focus during online classes and self-study sessions. Teachers and parents may also encourage students to track their screen time and avoid distractions. In workplaces, employees can use the extension to improve time management and increase efficiency during office hours. The project also has applications in digital wellness. Excessive screen time and social media addiction are growing concerns in modern society. This extension helps users become more conscious of their internet usage and encourages healthier browsing habits. By reducing unnecessary browsing time, users can achieve a better balance between work, study, and personal life. The project can be further enhanced by adding features such as cloud synchronization, user authentication, AI-based productivity suggestions, Pomodoro timers, and cross-browser support. In conclusion, the Productivity Tracker Chrome Extension is an effective and practical solution for managing online time and improving focus. By tracking website usage, blocking distractions, and generating productivity reports, the system helps users build disciplined digital habits. This project provides meaningful real-world benefits in education, workplaces, and personal productivity management.
